# 📊 CVM Data Analysis

Projeto de análise de dados da Comissão de Valores Mobiliários (CVM) com foco em extração de insights financeiros e identificação de padrões relevantes no mercado.

## 🚀 Objetivo

Explorar dados públicos da CVM para gerar análises que auxiliem na tomada de decisão e entendimento do comportamento de empresas e fundos no mercado brasileiro.

## 🛠️ Stack

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Requests (para consumo de APIs ou scraping)

## 📂 Funcionalidades

* Coleta automatizada de dados da CVM
* Tratamento e limpeza de dados
* Análise exploratória (EDA)
* Geração de relatórios e visualizações
* Identificação de padrões e tendências

## 📊 Exemplos de Análise

* Evolução de indicadores financeiros
* Comparação entre empresas/setores
* Detecção de anomalias em dados financeiros

## ▶️ Como executar

```bash
git clone https://github.com/schindlerfx/wallet_crypto.git
cd cvm-data-analysis
pip install -r requirements.txt
python main.py
```

## 📈 Resultados

O projeto permite visualizar dados estruturados e identificar tendências relevantes no mercado financeiro brasileiro.
